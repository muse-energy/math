## どうやって作ったか
MkDocs (Material for MkDocs) + KaTeX でやっています。

気になっている部分：

- 可換図式が(自由に)書けない
- 下付き文字のための記号`_`が数式内で正しく扱われないことがある気がする
- 他にも別行立て数式あたりで微妙に変な挙動がある気がするがイマイチよくわからない…
