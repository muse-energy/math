# 写像(1)

高校の数学では、二次関数(あるいは一般に$n$次間数)・三角関数・指数関数など、様々な関数を扱っていた。これらは、実数 $x$ を入力するとなんらかの実数 $f(x)$ を出力するものだと考えることができる。

また、平面を反時計回りに90°回転させるという操作を考えてみよう。この操作は、点 $P(x, y)$ を入力すると別の点 $Q(-y, x)$ を出力するものだと考えることができる。

これらのように、入力に対してなんらかの操作がほどこされて出力が得られる、という状況は数学では非常に多い。このような状況は、**写像**という概念を用いて一般的に表すことができる。

!!! definition "定義(写像)"
    $A$, $B$ を集合とする。$A$ の元 $a$ に対して、$B$ の元 $f(a)$ を定めるような規則 $f$ のことを、$A$ から $B$ への**写像**という。このとき、$A$ のことを $f$ の**定義域**とよび、$B$ のことを $f$ の**終域**と呼ぶ。また、このとき、$f: A \to B$ とか、$A \overset{f}{\to} B$ のように書く。

写像の例をいくつか挙げる。

!!! example "例"
    - 普通の二次関数 $f_1(x) = x^2 $ は、実数 $x$ に対して実数 $x^2$ を定める。すなわち、$f_1$ は$\mathbb{R}$ を $\mathbb{R}$ への写像である。
    - 同様に、$f_2(x) = \sin x$, $f_3(x) = \cos x$, $f_4(x) = e^x$ はいずれも$\mathbb{R}$ から $\mathbb{R}$ への写像である。
    - $f_5(x) = 1/x$ は、 $x=0$ のときに定義されていないので、$\mathbb{R}$ から $\mathbb{R}$ への写像とは言えない。
        - $\mathbb{R}^* := \mathbb{R} \setminus \{0\}$ (差集合)とおくと、$f_5$ は $\mathbb{R}^*$ から $\mathbb{R}$ への写像である。
    - 座標平面上の三角形全体の集合を $T$ とおく。三角形 $\triangle ABC$ に対してその面積 $S(\triangle ABC)$ を対応させることにより、写像 $S: T \to \mathbb{R}$ が得られる。
