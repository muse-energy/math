# 数学と論理

数学は「数の学問」であるが、このノートで扱うような現代的な数学は「論理と集合の学問」といってもよい。
数学では、数だけでなく、直線や円などの様々な図形、関数や方程式など、非常に多くの概念が扱われる。
これらの概念は、実はすべて**集合**を用いて定式化することができる。

さらに、数学では数や図形や関数について、様々な性質を考え、それを証明することがある。例えば、「12は4の倍数である」とか、「円 $x^2+y^2 = 1$ は領域 $x + y \leq 10$ に含まれる」といった具合である。これらの性質を扱うには、**論理**に関する知識を用いる。

解析学・幾何学・代数学といった多くの専門的な数学の分野では、論理や集合の言葉が使われている。そのため、論理や集合について学ぶことは、専門的な数学を学ぶための第一歩となるだろう。
